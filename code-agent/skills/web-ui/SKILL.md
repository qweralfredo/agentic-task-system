# Web UI

Priorities:

1. Keep operator actions obvious.
2. Show loading, success and error states explicitly.
3. Make desktop and mobile layouts usable.
4. Surface context like files, commands and skills without clutter.

See [references/panels.md](references/panels.md) for dashboard guidance.
